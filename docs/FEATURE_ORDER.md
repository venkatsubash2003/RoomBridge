# Feature implementation order

The prototype presents the features in the following product order:

1. Secure authentication and optional institutional verification
2. Housing-needs and lifestyle onboarding
3. Weighted roommate matching and deal-breakers
4. Property and room listings
5. Search, commute filters, saved searches, and alerts
6. Private and group messaging
7. Lease replacement and landlord approval workflow
8. Prospective household formation and rent planning
9. Safety, reporting, moderation, and privacy controls
10. International-user guidance, languages, currencies, and communities
11. Financial and household tools
12. Post-move agreements, chores, maintenance, and reminders
13. Administrative moderation, verification, analytics, fraud, and support tools

## Production delivery phases

### Phase 1 — Foundation

Backend API, database migrations, secure authentication, authorization, audit logging, file storage, and automated tests.

### Phase 2 — Marketplace

Profiles, matching, listings, search, maps, saved searches, notifications, reporting, and moderation.

### Phase 3 — Communication and households

Real-time messaging, translation, calls, household groups, voting, lease handoffs, and secure documents.

### Phase 4 — Living and financial services

Agreements, expenses, reminders, maintenance, compliant payments, insurance integrations, and advanced analytics.
